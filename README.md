# Biodiversidad-Argentina-An-lisis-de-Big-Data
Apache Spark, PySpark, Python, Manejo de Big Data, Limpieza y Transformación de Datos, Análisis Ecológico, Folium. Se aplicó Spark para el análisis escalable de más de 79,000 registros biológicos. Se identificó la estructura de la biodiversidad (dominio taxonómico) y se visualizaron zonas de mayor concentración de vida en un Mapa Interactivo.
